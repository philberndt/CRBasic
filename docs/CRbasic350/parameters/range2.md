# Range

The expected input from the sensor. To choose the appropriate range, multiply the expected current by 100(ohms).

Right-click the parameter to display a list or enter the code directly.

| Code   | Description     |
| ------ | --------------- |
| mV2500 | -100 to 2500 mV |
| mV34   | -34 to +34 mV   |

Type: Constant
