# Units

For the Delay instruction, the Units parameter is the time units for the delay. A numeric code is entered. Right-click the parameter to display a list.

| Code | Alphanumeric Code | Description  |
| ---- | ----------------- | ------------ |
| 0    | usec              | microseconds |
| 1    | msec              | milliseconds |
| 2    | sec               | seconds      |
| 3    | min               | minutes      |
| 4    | hr                | hours        |
| 5    | day               | days         |

Type: Constant
