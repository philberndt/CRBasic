# TimeOpt (Timer Action)

The action on the Timer. The timer instruction returns the value of the timer after the action is performed. A numeric code is entered. Right-click the parameter to display a list.

| Code | Description     |
| ---- | --------------- |
| 0    | Start           |
| 1    | Stop            |
| 2    | Reset and start |
| 3    | Stop and reset  |
| 4    | Read only       |

Type: Constant
