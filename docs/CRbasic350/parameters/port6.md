# Port

Specifies which digital port pair to use for monitoring sensor channels A and B, respectively. Valid ports are:

| Code | Description                   |
| ---- | ----------------------------- |
| C1   | Control ports 1 and 2         |
| SE1  | Single-ended channels 1 and 2 |
| SE3  | Single-ended channels 3 and 4 |

Type: Constant
