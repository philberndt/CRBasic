# URL Escape Codes

| Character | URL Escape Code |
| --------- | --------------- | ---- |
| SPACE     | %20             |
| <         | %3C             |
| >         | %3E             |
| #         | %23             |
| %         | %25             |
| +         | %2B             |
| {         | %7B             |
| }         | %7D             |
|           |                 | % 7C |
| \         | %5C             |
| ^         | %5E             |
| ~         | %7E             |
| [         | %5B             |
| ]         | %5D             |
| '         | %60             |
| ;         | %3B             |
| /         | %2F             |
| ?         | %3F             |
| :         | %3A             |
| @         | %40             |
| =         | %3D             |
| &         | %26             |
| $         | %24             |
