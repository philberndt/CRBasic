# SW5Port (Switched 5V Port)

The switched 5 volt port to use with this instruction. The number of available ports depends upon the CDM being programmed. Right-click the parameter for a list of valid options.

| Code | Description   |
| ---- | ------------- |
| 1    | Switched 5V 1 |
| 2    | Switched 5V 2 |
| 3    | Switched 5V 3 |
| 4    | Switched 5V 4 |

Type: Constant
