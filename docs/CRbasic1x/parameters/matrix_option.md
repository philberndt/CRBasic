# Option

Specifies the matrix operation to perform with the two-dimensional arrays, A and B. The Option can be:

| Alphanumeric | Description           |
| ------------ | --------------------- |
| 1            | Result = A + B        |
| 2            | Result = A - B        |
| 3            | Result = A \* B       |
| 4            | Result = Transpose(A) |
| 5            | Result = Inverse(A)   |
