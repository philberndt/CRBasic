# SDIPort (SDI Port)

The port to which the SDI-12 sensor is connected. Right-click the parameter to display a list. A numeric value is entered.

| Code | Description    |
| ---- | -------------- |
| C1   | Control Port 1 |
| C3   | Control Port 3 |
| C5   | Control Port 5 |
| C7   | Control Port 7 |

Type: Constant
