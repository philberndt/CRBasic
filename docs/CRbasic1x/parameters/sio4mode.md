# SIO4Mode

Defines which port the instruction will affect.

| Code | Description                |
| ---- | -------------------------- |
| 1    | Send/Receive Port 1        |
| 2    | Send/Receive Port 2        |
| 3    | Send/Receive Port 3        |
| 4    | Send/Receive Port 4        |
| 5    | Send to all Ports (global) |
