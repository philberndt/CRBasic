# Interface (Device)

Selects the device to be read by the function. Right-click the parameter for a drop-down list of options:

| Code | Description         |
| ---- | ------------------- |
| 0    | Ethernet            |
| 1    | PPP                 |
| 2    | CS I/O Interface(1) |
| 3    | CS I/O Interface(2) |
| 5    | USB                 |

Type: Constant or Variable
