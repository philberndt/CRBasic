# DSTnEnd (Daylight Saving Time Occurrence of Day End)

Specifies the occurrence of the day that daylight saving ends. Right-click the parameter to display a list of options:

| Code | Occurrence |
| ---- | ---------- |
| 1    | First      |
| 2    | Second     |
| 3    | Third      |
| 4    | Fourth     |
| 5    | Last       |

Type: Constant, integer, or variable
