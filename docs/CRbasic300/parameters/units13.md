# Units

The units for the interval parameter. Right-click the parameter to display a list.

| Code | Description                    |
| ---- | ------------------------------ |
| msec | milliseconds                   |
| sec  | seconds                        |
| min  | minutes                        |
| hr   | hours                          |
| day  | days                           |
| mon  | month (seconds into the month) |

Type: Constant
